# 🤖 LinguaAI — NLP Assistant Chatbot

## 📌 Introduction

LinguaAI is an advanced AI-powered chatbot designed to demonstrate real-world applications of **Natural Language Processing (NLP)**. It enables users to interact through natural language queries and receive structured, intelligent, and real-time responses.

The system combines a **pre-built NLP knowledge base** with optional **AI API integration**, making it both educational and interactive for learning core NLP concepts.

---

## 🚀 Features

* 💬 Interactive chatbot interface (ChatGPT-style UI)
* 🧠 Built-in NLP knowledge base (instant responses without API)
* 🔑 API integration support (Anthropic / AI models)
* 🌐 Real-time conversational experience
* 📝 Markdown-supported responses (code, tables, formatting)
* 📚 Covers key NLP topics (BERT, GPT, TF-IDF, NER, etc.)
* 📂 Chat session management with sidebar

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **AI Integration:** Anthropic API (optional)
* **Concepts Used:** NLP, Transformers, Tokenization, Sentiment Analysis

---

## 📋 Requirements

* A modern web browser (Chrome, Edge, Firefox)
* Internet connection (only required for API-based responses)

---

## ⚙️ How to Run

1. Download or clone the repository

```bash
git clone https://github.com/your-username/LinguaAI-Assistant.git
```

2. Open the project folder

3. Run the chatbot

* Open `nlp-chatbot.html` in your browser

---

## 🔑 API Setup (Optional)

* Enter your **Anthropic API key** in the sidebar
* Enables full AI-powered responses
* Without API → chatbot still works using built-in NLP knowledge
